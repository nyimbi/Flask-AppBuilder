Getting Started

Welcome to the Flask-AppBuilder Enhanced Widget Library!

This library provides modern, feature-rich widgets to enhance your Flask-AppBuilder applications.

Installation
# Widgets are included with this Flask-AppBuilder installation
from flask_appbuilder.widgets import ModernTextWidget, JSONEditorWidget
Widget Categories
  • Modern UI: Enhanced versions of standard widgets
  • Advanced Forms: Complex form building and validation
  • Specialized Data: Widgets for JSON, arrays, and complex data
Basic Widget Usage

Use widgets in your WTForms forms:

from flask_appbuilder.widgets import ModernTextWidget
from wtforms import StringField, Form

class MyForm(Form):
    name = StringField('Name', widget=ModernTextWidget(
        icon_prefix='fa-user',
        show_counter=True,
        max_length=100
    ))
Form Integration

Integrate widgets with Flask-AppBuilder forms:

from flask_appbuilder.forms import DynamicForm

class MyModelForm(DynamicForm):
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        
        # Apply widgets to specific fields
        if 'description' in self._fields:
            self._fields['description'].widget = ModernTextAreaWidget(
                rich_text=True,
                markdown_preview=True
            )
ModelView Integration

Use widgets in ModelView classes:

from flask_appbuilder.views import ModelView

class MyModelView(ModelView):
    datamodel = SQLAInterface(MyModel)
    
    add_form_widget = {
        'json_data': JSONEditorWidget(show_tree_view=True),
        'tags': TagInputWidget(max_tags=10)
    }
    
    edit_form_widget = {
        'json_data': JSONEditorWidget(show_tree_view=True),
        'tags': TagInputWidget(max_tags=10)
    }
Widget Customization

All widgets support extensive customization:

# Example: Customizing ModernTextWidget
widget = ModernTextWidget(
    icon_prefix='fa-envelope',      # Add icon
    icon_suffix='fa-check',         # Suffix icon
    show_counter=True,              # Character counter
    max_length=255,                 # Max length
    autocomplete_source='/api/suggestions',  # Autocomplete
    floating_label=True             # Floating label style
)
Common Issues
Widget not displaying correctly
Ensure Bootstrap CSS/JS are loaded in your template
JavaScript errors
Check browser console for specific errors. Some widgets require modern browsers.
Form validation issues
Make sure field names match your model attributes
Performance issues
Large datasets should use pagination or lazy loading options
